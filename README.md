# Cyber Password Security

Simple password strength analyzer written in Python. The goal of this project is to demonstrate basic security practices related to password creation.

## Features

- Checks password length
- Checks lowercase and uppercase letters
- Checks digits
- Checks special characters
- Returns a final strength rating (Weak / Medium / Strong)

## How to run

1. Make sure you have Python 3 installed.
2. Run in the project directory:

```bash
python password_tool.py
```

## Example

Input:
Enter password to analyze: MyPass123!

Output:
Score: 5/6
Strength: Strong
